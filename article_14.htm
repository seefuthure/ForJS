git add命令拓展

一、前言：

我们在之前有总结了 git常用命令总结，我在实际操作过程中，发现git add命令还有一些细节没有概括到，所以在这里补充下。如有不足或者错误之处，欢迎批评指正。


二、Git add 拓展：

我们知道，如果在项目中改动或者新增了某些文件，只要“ git add 文件名 ”即可。但是如果有多个文件的时候，我们难道一个一个的add吗？

下面就直接附上拓展命令：

git add -A .      //一次性添加所有改变的文件（注意 -A 选项后面还有一个句点）
git add -A		  //添加所有内容
git add .         //添加新文件和编辑过的文件，但不包括删除的文件
git add -u        //添加编辑或者删除的文件，但不包括新增加的文件
git add -h        //查看帮助

相信有了这几个命令，你使用git添加文件到暂存区的时候会更加方便，很简单的，赶紧动手实践下吧！
